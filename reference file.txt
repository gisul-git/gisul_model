After unzipping, in WSL terminal:

**1. Install dependencies**
```bash
pip install fastapi uvicorn transformers torch accelerate bitsandbytes cachetools pydantic streamlit requests pandas
```

**2. Start the backend**
```bash
uvicorn qwen_model_server_script:app --host 0.0.0.0 --port 8000
```
Wait until you see `Model loaded` in the logs.

**3. In a new WSL terminal, start the frontend**
```bash
streamlit run app.py
```

**4. Open browser**
```
http://localhost:8501
```

---

**If the other machine has no GPU**, add this before starting the backend:
```bash
export CUDA_VISIBLE_DEVICES=""
```
But the model will be very slow on CPU.
